
Aufgabe: read write

Öffne die Python-Dokumentation unter

https://docs.python.org/3/

Versuche herauszufinden, wie man eine Datei in Python öffnet.
Schlage außerdem nach, wie man Text aus einer Datei ausliest oder in eine Datei schreibt.

Tipp: Suche nach Informationen zur Funktion "open()".

1. Lesen
Schreibe ein Python-Skript, welches den Inhalt der Datei "text.txt" ausgibt.
Teste dein Skript mit einer Beispieldatei.

2. Schreiben
Schreibe ein Python-Skript, welches die Zahl "42" in die Datei "text.txt" schreibt
und nutze das Skript aus 1. um dein Programm zu testen.
